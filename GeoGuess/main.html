미니게임 · Kakao 로드뷰 기반

게임 개요

무작위로 선택된 국내 장소의 로드뷰를 보고, 오른쪽 아래 미니 지도에서 정확한 위치를 찍어 맞추는 게임입니다. 총 5라운드 진행되며, 실제 위치와의 거리(km)에 따라 점수를 획득합니다.

플레이 방법

  1. 로드뷰 화면을 회전/확대해서 주변 단서를 살펴봅니다.
  2. 화면 오른쪽 아래의 지도를 클릭해 추측 위치를 선택합니다.
  3. 거리와 획득 점수가 표시되고 추측 위치와 정답 위치가 표시됩니다.
  4. 5라운드 합산 점수가 최종 점수입니다.

조작

  • 로드뷰 이동: 마우스 드래그 / 클릭
  • 확대·축소: 마우스 휠
  • 지도에서 선택: 위치를 한 번 클릭
  • 브라우저 새로고침: Ctrl + F5

점수 체계

최대 점수는 5000점이며, 100km 이상 떨어진 경우 0점입니다. 100km 이내의 경우, 거리와 점수는 로그 함수를 기반으로 계산되며, 가까울수록 점수가 급격히 높아집니다.

거리(km)점수 계산
100km 이상0점
100km 미만5000 × [ 1 − ( log10(거리 + 1) ÷ log10(101) ) ]
※ 가까울수록 높은 점수를 부여하며, 점수 감소 폭은 거리 증가에 따라 완만해집니다.

알림

  • 일부 지역은 로드뷰가 제공되지 않아 다른 장소로 자동 대체될 수 있습니다.
  • 게임 중 표시되는 좌표는 서버에 저장하지 않습니다.
  • 모바일 환경에서는 데이터 사용량에 유의하세요.
  • 브라우저를 다크모드로 설정했을 시 일부 버튼 글자가 보이지 않습니다. 화이트로 설정해주세요.
